01
Canva 초기 세팅 순서
01
canva.com 접속 → 가입
구글 계정으로 가입. 무료 플랜으로 시작.
02
새 디자인 → 사이즈 직접 입력
"디자인 만들기" 클릭 → 사용자 지정 크기 → 아래 값 입력
Width : 1080px Height : 1920px 용도 : Instagram Reels / TikTok
03
배경 색상 설정
배경 클릭 → 색상 코드 직접 입력 → #0A0A0A
04
폰트 설정
제목 폰트 : Noto Sans KR Black 라벨 폰트 : DM Mono
05
완성 후 → 템플릿으로 저장
우측 상단 "…" → 템플릿으로 저장. 다음부터 복제해서 텍스트만 교체.

04
매일 교체 항목 — 딱 5개만
Type B — 카드뉴스 교체 항목
① 수치 → 오늘의 핵심 숫자 (예: 70%, $62.3B) ② 제목 → 해석 제목 2줄 (예: 나스닥 상승이 / 7개 종목에서 나왔다) ③ 출처 → *출처: Bloomberg · 2026.03.10 ④ 해석 텍스트 → 왜 이런 일이 일어났는지 2~3줄 ⑤ 날짜 → 하단 날짜 업데이트
Type C — 발언 인용 교체 항목
① 인물 이름 → 젠슨 황 / 워런 버핏 / 레이 달리오 등 ② 직함 → NVIDIA CEO / Berkshire CEO 등 ③ 인용 발언 → 핵심 발언 2~3줄 (텍스트 인용) ④ 날짜·출처 → 발언 날짜 + 출처 ⑤ 해석 텍스트 → BRIIQ 해석 2줄
작업 시간 목표 — 템플릿 복제 → 텍스트 5개 교체 → 내보내기 = 5분
하루 3개 기준 총 15분. 나머지는 전부 AI가 합니다.
